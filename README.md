# SQL Select Lab

Hey there! This is my SQL select lab where I'm learning to work with databases.

I practiced writing SQL queries to get data from employee tables in a company database. Stuff like selecting specific columns, renaming them with aliases, using CASE statements to categorize things, and even manipulating text and dates.

The main code is in `main.py`. You can run it with `python3 main.py` to see the queries and results.

I used sqlite3 to connect and run the SQL stuff.

Pretty cool to see how SQL can pull exactly the data you need!
